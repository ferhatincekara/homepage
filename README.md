## Getting Started

My personal website

### Prerequisites

- Node.js
- Npm or Yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ferhatincekara/homepage.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. To Start Server:
   ```sh
   yarn dev
   ```
4. To Visit App:
   ```sh
   localhost:3000
   ```
